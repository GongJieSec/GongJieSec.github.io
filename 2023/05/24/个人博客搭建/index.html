个人博客搭建

发表于2023-05-24|更新于2023-07-03|Blog
|阅读量:

  hellow， 大家好！我是Gong Jie 这个我的第一篇博文，为什么要搭建这个博客呢？第一嘛是进行知识的整理，第二嘛也是方便自己回顾知识点，当然还有最最重要的一点，那就是交一些志同道合的盆友，本站搭建于2023年5月24日，耗时三天，自己整理了本站的搭建流程及美化过程，欢迎交流学习（欢迎白嫖）

创建相关目录

打开此电脑—>进入D盘根目录—>右键—>新建文件夹—>个人博客
进入 D:\个人博客 目录，新建如下文件夹：

  • Git    -Git程序
  • Blog    -博客内容
  • VScode  -博客编辑器
  • Nodejs  -博客运行环境

安装相关环境

点击Git进入下载页面
选择Download for Winodws下载64-bit Git for Windows Setup
下载完成后打开Git-2.40.1-64-bit.exe安装程序
选择安装目录D:\个人博客\Git
Next完成安装

点击Node.js进入下载页面
选择18.16.0长期维护版下载安装程序
打开node-v18.16.0-x64.msi安装程序
选择安装目录D:\个人博客\Nodejs
Next完成安装

点击VScode进入下载页面
选择User Install x64下载安装程序
下载失败的小伙伴可以直接在Windeos Store下载安装
打开VSCodeUsersetup-x64-1.78.2.exe安装程序
选择安装目录D:\个人博客\Vscode
Next完成安装

安装完成后打开cmd窗口依次输入下面命令，能够回显相关版本号，则安装成功。

创建Github仓库

GitHub 主页右上角加号 -> New repository
Repository name中输入 “用户名.github.io”
勾选 Initialize this repository with a README
点击 Create repository完成创建
